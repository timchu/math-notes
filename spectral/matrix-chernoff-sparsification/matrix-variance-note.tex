Matrix variance as defined in Matrix Bernstein is the same as regular variance.

\begin{align}
  E((A+B)^2 - E((A+B)^2) = 
  E (A^2 + B^2 + AB + BA) - E(A^2) - E(AB+BA) - E(B^2)
  = Var(A) + Var(B) + E(AB + BA) - E(AB+BA)
  E (XY) = E(X)E(Y)
\end{align}
